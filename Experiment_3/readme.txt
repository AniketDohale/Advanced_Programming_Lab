Functions and Generators

# Content
-> Function Basics: def statement, definition and call, Local Variables
-> Function Scope: the LEGB rule, Global Statement, nested functions
-> Function Arguments: passing arguments, special arguments

# Aim
Write a Merge-Sort algorithm using the concept of Divide-and-Conquers to arrange the elements in ascending orders. 
Use all the operations of divide and conquers as functions.
