# Aim:
Write a Menu-driven Text Applications to solve Five Problems as a Menu-driven Text-based Application.
It presents the User with a Set of Choices.
1) Sum of Input Numbers
2) Average of Input Numbers
3) Mean of Input Numbers
4) Median of Input Numbers
5) Mode of Input Numbers
X) Quit
The User makes a Selection, which is then Excuted.
The Program exits when the User Chooses the "Quit" option.
The Great Advantage of a Program like this it Allows the User to run as Many iterations of your Solutions without Necessarily
having to restart the Same Program Over and Over Again.
