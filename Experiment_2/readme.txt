Derived Data Types

# Content:
-> Built-in Derived Data Types: Lists, Tuples, Dictionaries, Strings, Sets and Methods of these Data Types
-> Errors and Exception Handling: try block

# Aim:
Write a Menu-driven Text Application to maintain Bank Accounts of Customers using Lists, Dictionary, Strings and Sets.
This Application handles Error and Exception using try block.
The Application allows following Operations as Python Functions:
1) Adding and Delete Accounts of Customers.
2) Deposit and Withdraw Money to/from Accounts.
3) List Last 3 Transactions on Particular Accounts.
4) Exit from the Application.
